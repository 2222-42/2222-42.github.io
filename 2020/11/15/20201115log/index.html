isabelle

証明すること:

fold c empty ~ c

求められること：

  • empty tableから任意のtableに一般化する

  • tに対するside conditionの追加

  • conditionを除いてコマンドが等しい

    • commandの等しさは、
      • 全てのstateではなく、
      • preconditionを満たすstateについて、
      • 実行が一致することが求められる

equiv_up_to:

p |= c ~ c'
    = 
ALL s s'. P s --> (c,s) => s' = (c', s) => s'

bequiv_up_to:

p |= b <~> b'
    = 
ALL s . P s --> bval b s = bval b' s
